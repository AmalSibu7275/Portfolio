Project Overview
"Guess the Number – Horror Edition" is a web-based interactive game built as a creative twist on the classic number guessing game. Designed with a hand-drawn, horror-themed aesthetic, the project was developed to enhance both front-end development skills and user engagement through visual storytelling and suspense-driven gameplay.

Key Features
Interactive Gameplay: The user attempts to guess a randomly generated number within a given range. Each wrong guess triggers horror-themed feedback (e.g., eerie sounds, flickering screen effects, or creepy animations).
Hand-Drawn Visual Style: The UI embraces a sketchy, illustrated horror art style to create a unique and immersive atmosphere. Assets and UI elements were custom-styled to reflect a haunted or eerie theme.
Responsive Design: Built to run smoothly on desktop and mobile devices, adapting to various screen sizes while maintaining atmosphere and gameplay clarity.
JavaScript Logic: Core gameplay—including random number generation, input validation, and result evaluation—is implemented in clean, modular JavaScript.
User Feedback: Players receive dynamic responses based on their guesses, including jump scares or narrative messages to maintain tension.
Version Control: Tracked using Git for code collaboration and history management.

Learning Outcomes
Strengthened understanding of JavaScript logic, DOM manipulation, and event handling.
Applied responsive design principles and CSS animation techniques.
Gained experience in integrating aesthetic themes into functional code.
Collaborated on version control using Git with another developer (Amal S.).

Challenges Overcome
Balancing gameplay logic with thematic visual elements without overwhelming the user.
Ensuring performance optimization across devices despite animated and stylistic assets.
